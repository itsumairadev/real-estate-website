# Real Estate Website

A modern, responsive real estate website with advanced features.

## Features
- Advanced property search
- Filtering by type, price, bedrooms
- Virtual property tours
- Agent profiles
- Contact forms
- Newsletter subscription

## Technologies Used
- HTML5
- CSS3
- JavaScript
- Font Awesome Icons
- Google Fonts

## Live Demo
https://real-estate-website-nine-lime.vercel.app/

## How to Run
1. Clone this repository
2. Open index.html in browser
